# Instrucciones 
Hay dos formas de ver contenido de este trabajo:

El primero es ir al apartado code de este repositorio y descargar el ZIP, una vez guardado se descomprime y abre la carpeta que se generó. Al final, abres el index.html en tu navegador de preferencia.

La segunda, es solo ingresar al siguiente link, que te llevará al contenido inicial por medio de GitHub Pages:

[https://oswaldomh10.github.io/pruebaTecnicaEstadia/index.html](https://oswaldomh10.github.io/pruebaTecnicaEstadia/index.html)
